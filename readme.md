## Project Objective
The goal of this project is to detect the presence of groundwater in the subsurface.  
The main steps are:

1. Create a prototype graph to visualize the subsurface.  
2. Build a model to detect water and suggest the best areas to explore first (in order).

---

## Project Structure

### 1. Code
- Purpose: Data generation
- Technology: Octave
- Steps:
  - Create random data
  - Plot the image based on the generated data

### 2. Data
- Purpose: Store generated random data from the `Code` folder
- Format: CSV

### 3. Plot
- Purpose: Store example graphs generated from the random data and the model
- Format: PNG

### 4. Model
- Purpose: Groundwater detection
- Technology: Python
- Environment: Jupyter Notebook
