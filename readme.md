## Project Objective

The objective of this project is to detect the presence of groundwater in the subsurface using data visualization and computer vision techniques.  
The project follows a two-step approach:

1. Generate and visualize subsurface resistivity data through prototype plots.  
2. Analyze the generated plots using an image processing model to detect groundwater and rank the most promising areas for exploration.

---

## Project Structure

The project is organized into the following directories:

### Code
- **Purpose**: Synthetic data generation and visualization  
- **Technology**: GNU Octave  
- **Description**:
  - Generates random resistivity data simulating subsurface conditions
  - Produces 2D and 3D resistivity plots from the generated data

### Data
- **Purpose**: Store generated resistivity data  
- **Format**: CSV  
- **Description**:  
  Contains the synthetic datasets generated by the scripts in the `Code` directory.

### Plot
- **Purpose**: Store generated visualization outputs  
- **Format**: PNG  
- **Description**:  
  Contains resistivity maps generated from synthetic data as well as annotated outputs from the groundwater detection model.

### Model
- **Purpose**: Groundwater detection and analysis  
- **Technology**: Python  
- **Environment**: Jupyter Notebook  
- **Description**:  
  Uses computer vision techniques to detect groundwater zones in resistivity plots and rank them based on surface area.


### Documentation
- **Purpose**: Technical report and user guide  
- **Format**: PDF or Word  
- **Description**:  
  Contains a comprehensive report describing the system architecture, workflow, implementation details of each module, and step-by-step instructions to reproduce the results.
